# First Aid Awareness Analysis - Alexandria Case Study 🇪🇬

This project explores the intersection of **Medicine** and **Computer Science** through an analytical study of first aid awareness in Alexandria, Egypt. 

Inspired by the **CS50** methodology, this work aims to bridge the gap between emergency medical response and algorithmic thinking.

## 📁 Project Contents
- `First_Aid_Awareness_Research_Mina_Sherif.pdf`: The full research paper (Bilingual: Original Arabic Data with Professional English Analysis).
- `first_aid_logic.py`: A Python-based Decision Tree script that simulates emergency medical responses based on the research findings.

## 🚀 Key Insights (The "Logic" of First Aid)
By treating community behavior as a **Data System**, the research identified:
- **79.5% Success Rate:** High awareness in bleeding management.
- **The "Broken Link":** Only **47.4%** awareness of the national emergency hotline (123), which represents a system failure regardless of individual skill.
- **Cognitive Debugging:** A need to correct the "behavioral algorithm" for fainting cases (37.5% incorrect practices).

## 🛠 Tech Stack
- **Data Analysis:** Field survey with 41 participants.
- **Programming:** Python (Logic & Decision Trees).
- **Research:** Academic Writing & Translation.
